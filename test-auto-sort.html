🚀 Tính năng mới: Auto-Sort by Filename

Sắp xếp tự động theo tên tập tin! Bây giờ bạn có thể sắp xếp tất cả ảnh trong gallery chỉ với một cú click.

🔧 Demo Gallery Controls Layout:

➕ Thêm hình ảnh 🗑️ Xóa tất cả
☰ Kéo thả để sắp xếp thứ tự
Sắp xếp tự động: 📊 A-Z 📊 Z-A

✨ Tính năng Auto-Sort:

  • Sắp xếp tự động theo tên tập tin A-Z
  • Sắp xếp tự động theo tên tập tin Z-A
  • Sắp xếp thông minh với localeCompare (hỗ trợ tiếng Việt)
  • Numeric sorting (ví dụ: image1, image2, image10)
  • Confirmation dialog trước khi sắp xếp
  • Visual feedback với thông báo tùy chỉnh
  • Tự động ẩn/hiện khi có/không có ảnh
  • Responsive design cho mobile

🎯 Cách sử dụng:

  1. Tạo gallery và thêm ít nhất 2 ảnh
  2. Nhấn nút A-Z để sắp xếp từ A đến Z
  3. Nhấn nút Z-A để sắp xếp từ Z đến A
  4. Xác nhận trong dialog popup
  5. Thứ tự sẽ được cập nhật tự động
  6. Lưu gallery để áp dụng thay đổi

🛠️ Cải tiến kỹ thuật:

  • Natural sorting với String.localeCompare()
  • Hỗ trợ Vietnamese locale
  • Numeric comparison option
  • Efficient DOM manipulation
  • Progressive enhancement
  • Event delegation

📱 Mobile Responsive:

  • Controls layout tự động điều chỉnh
  • Sắp xếp theo column trên mobile
  • Touch-friendly button sizing
  • Optimized spacing và gaps

🎨 Example Filename Sorting:

Before: zebra.jpg, apple.jpg, banana.jpg, 10-cat.jpg, 2-dog.jpg

After A-Z: 2-dog.jpg, 10-cat.jpg, apple.jpg, banana.jpg, zebra.jpg

After Z-A: zebra.jpg, banana.jpg, apple.jpg, 10-cat.jpg, 2-dog.jpg
